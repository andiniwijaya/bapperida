# Panduan Pengguna — BAPPERIDA

**Sistem Registrasi Penomoran dan Arsip Surat** · BAPPERIDA Kabupaten Bandung

## Masuk ke Sistem

1. Buka URL aplikasi yang disediakan institusi
2. Klik **Masuk**
3. Gunakan email atau username dan kata sandi
4. Centang **Ingat Saya** hanya pada perangkat pribadi yang aman

## Registrasi Akun Baru (Staff)

1. Klik **Daftar** di halaman login
2. Isi nama, username, email, bidang, kata sandi
3. Verifikasi email melalui tautan yang dikirim
4. Tunggu **persetujuan Super Admin**
5. Setelah disetujui, login kembali

## Dashboard

Menampilkan ringkasan aktivitas sesuai role: statistik surat, grafik, aktivitas terbaru, dan quick action.

## Registrasi Penomoran Surat

- Daftar registrasi penomoran surat keluar
- Tambah, ubah, cetak kartu penomoran (PDF)
- Filter tahun, bidang, jenis surat

## Arsip Surat Masuk / Keluar

- Catat dan kelola arsip surat
- Upload lampiran (PDF/dokumen sesuai batas sistem)
- Cetak, export PDF, Excel
- Soft delete dengan pelacakan

## Laporan

- Gabungan surat masuk dan keluar
- Filter periode, bidang, jenis
- Export PDF, Excel, cetak

## Notifikasi

- Ikon bell di header
- Klik notifikasi untuk membuka halaman terkait
- Tandai dibaca / tandai semua dibaca

## Profil Saya

- Ubah nama dan email
- Kelola kata sandi (menu Pengaturan)

## Tips

- Gunakan filter pada tabel untuk pencarian cepat
- Tombol reset filter mengembalikan filter ke awal
- Jika sesi habis, login ulang — pesan akan ditampilkan
- Untuk masalah teknis, hubungi administrator BAPPERIDA

## Hak Akses Staff

Staff umumnya hanya melihat data **bidang** yang ditugaskan. Admin dan Super Admin memiliki cakupan lebih luas sesuai kebijakan organisasi.
